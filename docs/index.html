Why RedundaNet?

A complete solution for distributed, encrypted storage

🔒

End-to-End Encryption

Your data is encrypted before it leaves your device. Only you hold the keys. Built on Tahoe-LAFS proven cryptography.

🌐

Mesh VPN Network

Nodes connect securely via Tinc VPN mesh network. No central server required. Works across NATs and firewalls.

📚

Redundant Storage

Data is split and distributed across multiple nodes. Configurable redundancy ensures availability even when nodes go offline.

🖦

Raspberry Pi Ready

Pre-built images for Raspberry Pi. Turn any Pi into a storage node in minutes. Perfect for home labs and edge deployments.

🐳

Docker Native

Fully containerized with Docker Compose. Easy deployment, updates, and management. Run anywhere Docker runs.

🤖

Python CLI

Modern Python CLI for node management. Initialize, configure, and monitor your network with simple commands.

Get Started

Choose your preferred installation method

🐳 Docker (Recommended)

Run RedundaNet using Docker Compose. Works on any system with Docker installed.

git clone https://github.com/adefilippo83/redundanet.git
cd redundanet
docker compose up -d

🐍 PyPI

Install the RedundaNet CLI tool via pip for node management.

pip install redundanet

redundanet init --name my-node
redundanet status

🖦 Raspberry Pi

Download our pre-built image, flash it to an SD card, and boot.

# Download from GitHub Releases
# Flash with Raspberry Pi Imager
# Boot and SSH in:
ssh redundanet@redundanet.local
# Password: redundanet
